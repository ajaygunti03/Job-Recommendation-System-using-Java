PROJECT ABSTRACT

The developed system is recommendation system for campus recruitment which helps college placement office to match company's profile and student's profile with higher precision and lower cost. for profile matching two matching methods are used.semantic matching,tree-based knowledge matching and query matching. these methods are integrated according to representations of attributes of students and companies and then the profile similarity degree is acquired. based on the profile similarity degree,preference lists of companies and student are generated.
